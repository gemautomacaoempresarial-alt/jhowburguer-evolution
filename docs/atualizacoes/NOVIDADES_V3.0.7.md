# AtenderBem v3.0.7

## Ações das mensagens

- Corrigido o desaparecimento da barra de ações ao mover o mouse do balão até os botões.
- Mantidos responder, selecionar, encaminhar, fixar, reagir e reenviar.
- Adicionado o botão de exclusão para mensagens enviadas pelo atendente.

## Exclusão

- **Excluir só para mim:** oculta uma ou várias mensagens apenas para o usuário atual.
- **Apagar para todos:** solicita a remoção da mensagem no WhatsApp e registra a exclusão no painel.
- Administradores e supervisores continuam vendo o registro apagado de forma esmaecida, com indicação de quem apagou.
- A seleção múltipla agora possui um botão de lixeira para exclusão em lote.
- Mensagens recebidas do cliente não podem ser apagadas para todos, mas podem ser ocultadas apenas para o usuário.

## Encaminhamento

- Modal reorganizado em grupos: WhatsApp, canais internos e pessoas da equipe.
- Adicionada busca por destino.
- Removida a rolagem horizontal e corrigido o alinhamento dos seletores.
